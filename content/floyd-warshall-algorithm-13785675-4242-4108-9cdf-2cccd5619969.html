Floyd-Warshall Algorithm

Introduction :

The Floyd-Warshall algorithm is a graph-analysis algorithm that calculates shortest paths between all pairs of nodes in a graph. It is a dynamic programming algorithm with O(|V|3) time complexity and O(|V|2) space complexity. 

The Floyd-Warshall algorithm works based on a property of intermediate vertices of a shortest path i.e. the algorithm is derived from a recurrence that the shortest path from vertex i to vertex j is minimum of { shortest path from i to j directly or shortest path from i to k and shortest path from k to j }.

An intermediate vertex for a path p = <v1, v2, ..., vj> is any vertex other than v1 or vj.(i.e. the initial and final vertex of a path)

Algorithm :

If the vertices of a graph G are indexed by {1, 2, ..., n}, then consider a subset of vertices {1, 2, ..., k}. Assume p is a minimum weight path from vertex i to vertex j whose intermediate vertices are drawn from the subset {1, 2, ..., k}. If we consider vertex k on the path then either:

            k is not an intermediate vertex of p (i.e. is not used in the minimum weight path)

            ⇒ all intermediate vertices are in {1, 2, ..., k-1} and the shortest such path has the length dijk-1

 

            k is an intermediate vertex of p (i.e. is used in the minimum weight path)

            ⇒ we can divide p at k giving two sub paths p1 and p2 giving vi ↝ k ↝ vj and the shortest such path has               the length dikk-1+ dkjk-1
