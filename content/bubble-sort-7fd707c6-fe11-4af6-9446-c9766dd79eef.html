Bubble Sort Algorithm

Input: Integer[]: A

Output: Integer[]: sorted by increasing order

for i←1 to A.size-1 do

for j←i + 1 to A.size do

if A[i] > A[j] then

tmp←A[i]

A[i] = A[j]

A[j] = tmp

end if

end for

end for

return A

 

Theory:

Bubble sort, sometimes incorrectly referred to as sinking sort, is a simple sorting algorithm that works by repeatedly stepping through the list to be sorted, comparing each pair of adjacent items and swapping them if they are in the wrong order. The pass through the list is repeated until no swaps are needed, which indicates that the list is sorted. The algorithm gets its name from the way smaller elements "bubble" to the top of the list. Because it only uses comparisons to operate on elements, it is a comparison sort. Although the algorithm is simple, most of the other sorting algorithms are more efficient for large lists.

Analysis:

To start with, Let us consider an array A[1..n] to be sorted. All the elements of A[] are distinct.

 

Case 1: For n=2

A[2]={6,5}

Since, 6 > 5, so we exchange the position of 6 with that of 5. The purpose of exchanging positions of two numbers after comparing their value is to move the bigger number backwards and bring the smaller number forward in the array.

 

Case 2: For n=3

A[3]={6,5,3}

We use the same approach, moving the bigger number backwards and bringing the smaller number forward. Since, 6 > 5, we exchange 6 with 5, the array becomes {5, 6, 3}.

Now we move forward with the comparisons and we compare 6 with 3. Since, 6 > 3, the array becomes {5, 3, 6}. In doing so, we have moved the largest number, that is 6, to the last position in the array in 3-1=2 comparisons and we need not consider this last position anymore.

Now, the problem is reduced to the case n=2 and array as {5, 3}. We apply sorting and array becomes {3, 5}. So the final result of case n=3 is the sorted array {3, 5, 6}.

 

Case 3: For n=k

A[k]={6,5,3,..,kth element}

Using similar approach, compare first two elements and sorting them, then comparing next two elements and so on, we are able to move the largest element at kth position after k-1 comparisons. Array is now reduced to A[1..k-1]. Again in k-1-1 i.e. k-2 comparisons largest element is moved to last position. We continue this process till array size is reduced to two, in which case the last comparison is done to obtain the final sorted array.

Total comparisons for k elements= k-1 + k-2 + … + 2 + 1.

Source: wiki

 

Hence, algorithm continuously steps through a list, swapping items until they appear in the correct order. The largest element gets sorted first, with smaller elements taking longer to move to their correct positions.

 

Conclusion:

To conclude, Bubble Sort Algorithm uses recursive method of sorting large array by breaking down the large array into small array s of size 2 and sorting elements until all elements are in ascending order.
