Algorithm Explanation:-

The Euclidean algorithm is used to calculate the greatest common divisor (GCD) of two natural numbers a and b.

The algorithm states that for two natural numbers a and b, such that a > b and if :-

r0 = a mod b

r1 = b mod r0

r2 = r0 mod r1

..

..

rk = rk-2 mod rk-1

and rk = 0 than GCD of a and b will be rk-1


Approaches might be used for solving the problem

Approach 1:

Before explaining the approach let's see two case of this problem. Consider a and b be two natural numbers such that a > b and b is a factor of a i.e. b mod a = 0. In this case GCD(a,b) will be equal to b.

Now let's consider the case in which b is not a factor of a i.e. a mod b != 0 in this case let GCD(a,b) be g.

If we consider the problem backwards, so g should exactly divide a mod b, as:

a = x*b + c,

So if g divides b completely then it will divide x*b also and since g also divides a completely we can state that :

g mod c is also = 0

which implies that GCD(b,c) is g. Now applying the above rule again on

b = x1*c + d

we get GCD(c,d) = g

In similar way the problem will go on reducing till the condition: GCD(g, xn*gn*g) where it wont be able to reduce further.

Now if in the whole process we change our assumption of g from known quantity to unknown quantity and analyse the pattern we are left with a relation

GCD(a,b) = GCD(b, a%b) = GCD(a%b, b%(a%b)) = .......... = GCD(g,xnn) = g

i.e the Euclidean Algorithm.


Approach 2:

In this approach first lets understand GCD of 2 numbers. GCD of two natural numbers a and b can be visualized as, if there is a floor of area a x b, then GCD of a and b will the length l of square tile with largest area which can exactly cover the whole floor.

Now if the square tile cover the whole floor then it means that l will divide a and b perfectly so square tile will also be able to cover floor with area b x b. So if we cover floor of a x b with b x b tiles (considering that b x b is already filled with l x l tiles)than we will be left with the area (a mod b) x b. And again since l x l tiles perfectly cover a x b  and b x b  so they will also perfectly cover  (a mod b) x b.

On recurring the above relation it will form a similar thing as Approach 1 explaining one of the possible thinking behind the discovery of Euclidean Algorithm.
